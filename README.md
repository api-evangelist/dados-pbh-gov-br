# Portal de Dados Abertos – Prefeitura de Belo Horizonte (dados-pbh-gov-br)

Portal de Dados Abertos – Prefeitura de Belo Horizonte is a open data portal open-data portal for Brazil running **CKAN**, exposing a standardized catalog API over ~588 datasets.
